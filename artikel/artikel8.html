OpenHMD: Proyek Sumber Kode Terbuka Untuk Pengembangan Virtual Reality (VR)





Pada era sekarang, hampir seluruh bagian pada kebutuhan komputasi terdapat alternatif yang menggunakan proyek sumber kode terbuka. Bahkan terdapat platform untuk kacamata Virtual Reality (VR) dan sejenisnya. Mari kita berkenalan dengan openHMD, sebuah proyek sumber kode terbuka untuk mengembangkan Virtual Reality.

Apa itu openHMD?

openHMD adalah sebuah proyek yang ditujukan untuk membuat sebuah Antarmuka Pemrograman Aplikasi atau lebih dikenal dengan sebutan API (Application Programming Interface) dan driver untuk teknologi yang mendalam. openHMD sendiri mendukung beberapa sistem operasi termasuk android, FreeBSD, Linux, openBSD, Mac OS dan Windows. Proyek ini mendukung perangkat keras seperti Oculus rift, HTC Vive, DreamWorld DreamGlass, Playstation Move, dan masih banyak lagi. Selain daripada itu, proyek ini juga mendukung berbagai bahasa pemrograman seperti Go, Java, .NET, Perl, Python dan Rust.

Perlu diketahui bahwa proyek ini berada dibawah lisensi Boost License.

Pembaharuan apa saja yang dimiliki openHMD?

openHMD sendiri sudah rilis sampai dengan versi 0.3.0 dan memiliki codename Djungelvral, sebuah nama tumbuhan akar manis dari Swedia. Berikut adalah fitur-fitur terbaru yang dimiliki openHMD:

  • 3Glasses D3
  • Oculus Rift CV1
  • HTC Vive and HTC Vive Pro
  • NOLO VR
  • Dukungan Windows Mixed Reality HMD
  • Deepoon E2
  • GearVR Gen1

    • Selain daripada itu, openHMD juga memberikan sebuah shader distorsi universal, dimana memungkinkan untuk hanya mengatur beberapa variabel pada driver yang memberikan informasi kepada shader mengenai ukuran lensa, penyimpangan berwarna, posisi, dan kebiasaan. Selain terdapat penambahan, tim pengembang openHMD juga melakukan beberapa penghapusan seperti dukungan untuk playstationVR yang diakibatkan oleh beberapa masalah dengan Windows dan mac OS karena header HID tidak lengkap.
